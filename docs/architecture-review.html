3. Material Duplication MEDIUM

Files: src/spacecraft/mod.rs, src/panels/mod.rs

// Materials recreated in every spawn function let wall_material = materials.add(StandardMaterial { ... }); let panel_material = materials.add(StandardMaterial { ... }); let seat_material = materials.add(StandardMaterial { ... }); // 12+ materials per spawn, never cached

Recommendation

  • Create MaterialCache resource with lazy initialization
  • Clone Handle<StandardMaterial> instead of recreating
  • Use Bevy's asset system: load from RON/JSON configs
